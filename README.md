# The Rehabilitation Ontology (Training)

Description: Training/simulation ontology for rehab cueing and skill training workflows.

More information can be found at http://obofoundry.org/ontology/rehabo-train

## Versions

### Stable release versions

The latest version of the ontology can always be found at:

http://purl.obolibrary.org/obo/rehabo-train.owl

(note this will not show up until the request has been approved by obofoundry.org)

### Editors' version

Editors of this ontology should use the edit version, [src/ontology/rehabo-train-edit.owl](src/ontology/rehabo-train-edit.owl)

## Contact

Please use this GitHub repository's [Issue tracker](https://github.com/manojniwas/rehabo-odk-training/issues) to request new terms/classes or report errors or specific concerns related to the ontology.
